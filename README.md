## Dotfiles Included

- **Neovim**: For productive coding.
- **Tmux**: Session and window management setup for a streamlined workflow.
- **Kitty**: Terminal emulator settings for aesthetics.
- **Cava**: Audio visualizer configuration to complement the terminal's look.
- **Neofetch**: System information display setup for a personalized terminal greeting.

## Installation

To install these configurations, clone the repository and symlink the files to the appropriate locations:

```bash
git clone https://github.com/yourusername/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
```

### Neovim

Link Neovim config:

```bash
ln -s ~/.dotfiles/nvim ~/.config/nvim
```

### Tmux

Link Tmux config:

```bash
ln -s ~/.dotfiles/tmux ~/.config/tmux
```

### Kitty

Link Kitty config:

```bash
ln -s ~/.dotfiles/kitty ~/.config/kitty
```

### Cava

Link Cava config:

```bash
ln -s ~/.dotfiles/cava ~/.config/cava
```

### Neofetch

Link Neofetch config:

```bash
ln -s ~/.dotfiles/neofetch ~/.config/neofetch
```
